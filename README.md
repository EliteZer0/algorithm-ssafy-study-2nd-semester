### 📅 진행 방식

매주 정해진 문제들을 각자 풀이<br>
본인의 폴더에 코드 작성 및 PR 생성<br>
주간 온라인/오프라인 미팅에서 코드 리뷰

### 📁 폴더 구조
```
알고리즘스터디/
├── member1/
│   ├── week1/
│   │   ├── problem1.java
│   │   └── problem2.java
│   └── week2/
│       ├── problem1.java
│       └── problem2.java
├── member2/
└── README.md
```

### 📌 제출 규칙

폴더명

본인 이름(또는 GitHub ID)으로 폴더 생성<br>
주차별 폴더 생성 (예: week1, week2, ...)


파일명

문제 번호와 제목을 포함<br>
예: boj_1234_문제제목.java, programmers_완주하지못한선수.java


커밋 메시지

[Week N] 문제이름 / 걸린시간<br>
예: [Week 1] BOJ 1234 괄호 / 30분
